Documento Completo de HTML y CSS

En este documento se explican los elementos básicos de HTML y CSS, incluyendo ejemplos de listas, formularios y tablas. Cada sección muestra el código y cómo se ve en la página.

1. Estructura Básica de HTML

<html>: Es el contenedor raíz de todo el documento.

<head>: Contiene metadatos (información sobre el documento) como el título y enlaces a archivos CSS o scripts.

<title>: Define el título que aparece en la pestaña del navegador.

<body>: Contiene el contenido visible, como encabezados, párrafos, imágenes, etc.

<h1>: Representa el encabezado principal, que generalmente se muestra con un tamaño de fuente grande y en negrita.

<p>: Define un párrafo de texto.

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la Página</title>
  </head>
  <body>
    <h1>Encabezado Principal</h1>
    <p>Este es un párrafo de ejemplo.</p>
  </body>
</html>

2. Listas

HTML permite crear listas ordenadas y no ordenadas para organizar información.

Lista No Ordenada (<ul>)

<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ul>

  • Elemento 2
  • Elemento 3

Lista Ordenada (<ol>)

<ol>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
  <li>Tercer elemento</li>
</ol>

  1. Primer elemento

3. Formularios

Los formularios permiten a los usuarios ingresar datos que pueden ser enviados y procesados en el servidor.

<form action="/submit">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre">
  <input type="submit" value="Enviar">
</form>

4. Tablas

Las tablas se utilizan para organizar datos en filas y columnas.

<table border="1">
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>23</td>
  </tr>
  <tr>
    <td>Juan</td>
    <td>30</td>
  </tr>
</table>

5. CSS Básico

CSS (Cascading Style Sheets) se utiliza para dar estilo a los elementos HTML. A continuación se muestra un ejemplo de CSS interno.

<style>
  body {
    background-color: lightgray;
  }
  h1 {
    color: blue;
  }
</style>

6. Resumen de Elementos HTML Básicos

<html>: Contenedor raíz de todo el documento.

<head>: Contiene metadatos y enlaces a recursos externos (CSS, scripts, etc.).

<title>: Define el título de la página (lo que se muestra en la pestaña del navegador).

<body>: Contiene el contenido visible de la página (textos, imágenes, formularios, tablas, etc.).

<h1>: Encabezado principal, generalmente se muestra con una fuente grande y en negrita.

<p>: Párrafo de texto.

<ul> y <ol>: Listas no ordenadas y ordenadas, respectivamente.

<li>: Elemento de una lista.

<form>: Define un formulario.

<label>: Etiqueta descriptiva para un campo del formulario.

<input>: Campo de entrada (por ejemplo, de texto, botón de envío, etc.).

<table>: Define una tabla.

<tr>: Define una fila en una tabla.

<th>: Define una celda de encabezado en una tabla.

<td>: Define una celda de datos en una tabla.

Con estos elementos básicos puedes construir y estructurar cualquier página web, aplicando estilos con CSS para mejorar su apariencia.
