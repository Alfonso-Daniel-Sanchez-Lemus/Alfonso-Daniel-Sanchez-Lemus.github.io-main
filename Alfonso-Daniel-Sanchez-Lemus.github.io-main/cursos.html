programación El nivel Exploratorio en programación introduce los fundamentos de la lógica de programación y los conceptos esenciales de codificación. Incluye el aprendizaje de estructuras de control básicas como bucles (for, while) y condicionales (if, else), así como la comprensión de tipos de datos primitivos (enteros, flotantes, cadenas de texto) y estructuras de datos simples (listas, arreglos). Este nivel suele enfocarse en un lenguaje de programación específico, como JavaScript
Inteligencia Artificial En esta aventura abordarás conocimientos desde la historia y Ciclo de Vida de una aplicación de aprendizaje de Máquina hasta la generación de datos sintéticos, para la creación de modelos IA.
Análisis de datos El nivel exploratorio de Análisis de Datos abarca conceptos fundamentales para entender y trabajar con datos. Incluye la recolección, organización y limpieza de datos, así como la utilización de herramientas básicas para la visualización de datos, como gráficos y tablas a través del lenguaje de programación Python.
Arquitectura en la nube Este entrenamiento te permitirá adquirir competencias técnicas en arquitectura en la nube, la cual es una habilidad clave en el mercado laboral actual.
Blockchain El nivel explorador en Blockchain se enfoca en la adquisición de conceptos clave y habilidades generales que permitan comenzar a navegar en esta tecnología de manera efectiva. Durante este proceso, se desarrollarán habilidades digitales como el manejo de plataformas online, la capacidad para investigar, buscar y gestionar información relevante, así como la disposición para aprender sobre tecnología y analizar sus aplicaciones y limitaciones. Además, se comprenderán aspectos fundamentales del Blockchain, como su funcionamiento como base de datos descentralizada, las criptomonedas, la seguridad criptográfica y la descentralización.
Ciberseguridad El nivel Explorador de ciberseguridad requiere comprender los conceptos fundamentales de la seguridad informática, incluyendo la confidencialidad, integridad y disponibilidad de la información. Es esencial conocer los tipos comunes de amenazas y ataques, como malware, phishing y ataques de fuerza bruta, así como las prácticas básicas de seguridad, como el uso de contraseñas fuertes, autenticación multifactor y mantener el software actualizado.
